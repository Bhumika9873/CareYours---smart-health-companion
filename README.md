# CareYours - Smart Health Companion

CareYours is a smart healthcare web application designed to help users manage their daily health activities efficiently.

## Features
- Medicine Reminder
- Health Tracking
- Wellness Monitoring
- User-friendly Dashboard

## Tech Stack
- React (Vite)
- Tailwind CSS
- Node.js (future integration)

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Goal
To build a smart assistant that helps users take better care of their health with ease.
